Pinoy Trivia Game

Tara na at maki chismis!

Project Type

Desktop Application

Technologies Used

Python, TKinter, Firebase

Brief Overview

Pinoy Trivia Game is a desktop application built with Python and Tkinter that highlights Philippine culture and trends through an interactive quiz format. Players have 1 minute and 30 seconds per round to answer questions and earn points across 7 different categories. The game features a leaderboard system, powered by Firebase, to track high scores and enhance competition.

Key Features

  • Philippine Culture & Trends Trivia
  • Timed Gameplay
  • Multiple Quiz Categories
  • Leaderboard System
  • Firebase Database Integration

Screenshots

Challenges & Solutions

Implementing real-time score tracking was challenging, but integrating Firebase allowed efficient data storage and leaderboard updates.

Outcome & Impact

The game promotes Filipino culture in an engaging way, encouraging players to learn while competing for high scores.
