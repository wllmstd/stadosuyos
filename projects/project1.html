CashMate

Effortless Cash Recognition for the Visually Impaired.

Project Type

Mobile App

Technologies Used

Python, TensorFlow, Flutter

Brief Overview

CashMate is an assistive mobile application designed to help visually impaired users recognize different banknotes through edge detection techniques.

Key Features

  • Real-time banknote detection
  • Audio feedback for recognition
  • Offline functionality

Screenshots

Challenges & Solutions

Ensuring high accuracy in edge detection was a challenge, which was solved by training the model with multiple datasets and enhancing feature extraction.

Outcome & Impact

Helped visually impaired users differentiate banknotes easily, with positive feedback from early testers.
