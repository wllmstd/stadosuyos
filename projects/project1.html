Brief Overview

CashMate is an assistive mobile application designed to help visually impaired users recognize different banknotes using object detection techniques. The app first utilizes the YOLOv8 model to detect and crop the banknote region. Then, the cropped banknote undergoes feature extraction using MobileNetV2, which determines whether the banknote is real or fake. To enhance user experience, CashMate includes a voice assistant named Cashy, providing easy navigation and accessibility within the app.

Key Features

  • Real-time banknote detection
  • Audio feedback for recognition
  • Money counter for total calculation
  • Digital wallet for representing money

Challenges & Solutions

Ensuring high accuracy in MobileNetv2 Model was a challenge, which was solved by training the model with multiple datasets and enhancing feature extraction.

Outcome & Impact

Helped visually impaired users differentiate banknotes easily, with positive feedback from early testers at SPED-Integrated School for Exceptional Children.
